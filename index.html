🌍 IP 국가 조회기

대량 IP 주소의 국가 정보를 조회하고 내보내기 (오프라인 지원)

내보낼 컬럼 선택:

조회 결과

IP 주소 국가명 (한글) 국가코드 대륙
IP 주소를 입력하고 "조회하기" 버튼을 클릭하세요.

📡 API 사용 방법

JavaScript로 프로그래밍 방식으로 사용할 수 있습니다:

// 단일 IP 조회 const result = CountryFinderAPI.lookup('8.8.8.8'); console.log(result); // { ip: '8.8.8.8', country: '미국', code: 'US', continent: '북아메리카' }
// 여러 IP 조회 const ips = ['8.8.8.8', '1.1.1.1', '208.67.222.222']; const results = CountryFinderAPI.lookupBulk(ips); console.log(results);
// Excel VBA에서 사용 (Internet Explorer 객체 사용) ' 웹 페이지를 로드하고 JavaScript 함수 호출 Dim IE As Object Set IE = CreateObject("InternetExplorer.Application") IE.Navigate "file:///C:/path/to/index.html" ' 페이지 로드 후 result = IE.Document.parentWindow.CountryFinderAPI.lookup("8.8.8.8")
